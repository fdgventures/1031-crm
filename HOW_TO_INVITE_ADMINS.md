# Как приглашать админов

## Пошаговая инструкция

### 1. Войдите в админ панель

1. Откройте http://localhost:3000/admin/signin
2. Войдите с учетными данными workspace_owner или platform_super_admin

### 2. Откройте страницу управления админами

- Нажмите на карточку **"Admin Management"** в дашборде
- Или перейдите напрямую: http://localhost:3000/admin/admins

### 3. Создайте приглашение

1. Нажмите кнопку **"+ Invite New Admin"**
2. Введите email будущего админа
3. Выберите тип роли:
   - **Platform Super Admin** - может управлять другими админами
   - **Admin** - обычный админ без прав управления
4. Нажмите **"Send Invitation"**

### 4. Скопируйте ссылку приглашения

После создания приглашения:

- Ссылка появится в поле ниже
- Нажмите **"Copy"** чтобы скопировать ссылку в буфер обмена
- Отправьте эту ссылку новому админу (через email, Slack, и т.д.)

### 5. Новый админ регистрируется

1. Новый админ открывает ссылку (напр: http://localhost:3000/admin/register-invite/abc123...)
2. Заполняет форму регистрации (email уже заполнен)
3. Создает пароль
4. Подтверждает email по ссылке из письма
5. После подтверждения становится активным админом

## Управление приглашениями

### Просмотр всех приглашений

На странице `/admin/admins` вы увидите:

- **Pending Invitations** - все открытые приглашения
- Для каждого приглашения показана ссылка
- Можно скопировать ссылку кнопкой **"Copy"**

### Удаление приглашений

- Только **workspace_owner** может удалять приглашения
- Кнопка **"Remove"** доступна только владельцу

## Права доступа

### Workspace Owner (fdgventures@gmail.com)

- ✅ Создание приглашений
- ✅ Удаление приглашений
- ✅ Просмотр всех админов

### Platform Super Admin

- ✅ Создание приглашений
- ❌ Удаление приглашений (только workspace_owner)
- ✅ Просмотр всех админов

### Admin

- ❌ Создание приглашений
- ❌ Управление админами

## Важные заметки

1. **Срок действия**: Приглашения действительны 7 дней
2. **Токен уникален**: Каждое приглашение генерирует уникальный токен
3. **Один email**: Нельзя создать два приглашения для одного email одновременно
4. **Уже зарегистрированные**: Если пользователь уже есть, приглашение не создастся

